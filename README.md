## 📈 Features

- **Real-time Rankings**: Stack ranking of engineers across both teams (Kiran Vijay's & Akanksha Saxena's)
- **Interactive Filters**: Filter by team, title, and search engineers
- **Customizable Weights**: Adjust metric weights to recalculate rankings in real-time
- **Visual Analytics**:

  - Individual radar charts
  - CSAT distribution
  - Monthly trend analysis
  - Team comparison metrics
- **Detailed Views**: Click any engineer to see detailed performance metrics and trends

## 📊 Metrics Tracked

- **CSAT Score** (Customer Satisfaction - case surveys only)
- **TTR** (Time to Resolution)
- **Closed Cases** (Total YTD)
- **SLA %** (Initial response met percentage)
- **Throughput** (Cases per week)
- **Reopen Rate** (Percentage of reopened cases)
- **Survey Count** (Surveys received)
- **Escalations** (Number of escalated cases)
